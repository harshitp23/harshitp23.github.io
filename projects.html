Student Grade Calculator (Java, MySQL, JDBC)

A full-stack desktop application designed to act as a persistent academic tracker. Unlike simple calculators, this system manages multiple courses and predicts future performance requirements.

  • Dynamic Architecture: Uses a hybrid database model that generates dynamic SQL tables for each subject to ensure data isolation.
  • Advanced Grading Logic: Supports both Weighted Grading (percentage-based) and Total Points modes, with automatic handling for extra credit.
  • Target Grade Predictor: An algebraic tool that calculates exactly what score is needed on future assignments to achieve a specific letter grade.
  • Real-Time Dashboard: Visualizes cumulative GPA and organizes courses by "Current" or "Completed" status.

Nexus Inventory - Full Stack Catalog (Spring Boot, MySQL)

A robust Full-Stack application designed to manage product inventories for e-commerce platforms. This project demonstrates a complete implementation of the Model-View-Controller (MVC) architecture.

  • Backend Engineering: Built with Java Spring Boot, utilizing Spring Security for authentication and Spring Data JPA for object-relational mapping.
  • REST API Design: Developed secure endpoints for full CRUD operations (Create, Read, Update, Delete) and custom SQL-based search queries.
  • Database Relations: Designed a normalized MySQL schema establishing One-to-Many relationships between Products and Categories.
  • Dynamic Frontend: Features a responsive "Admin Dashboard" UI built with HTML/CSS and JavaScript (Fetch API) that renders data dynamically and manages state without page reloads.
